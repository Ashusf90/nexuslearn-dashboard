# Next Gen Learning Dashboard

A futuristic animated student dashboard built using Next.js App Router, Supabase, Tailwind CSS, and Framer Motion.

## Features

- Bento grid dashboard layout
- Server-side Supabase data fetching
- Animated progress indicators
- Framer Motion stagger animations
- Responsive sidebar and mobile navigation
- Skeleton loading states
- Dark glassmorphism UI

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Supabase
- Framer Motion
- Lucide React

## Architecture

- Server Components used for Supabase fetching
- Client Components used for animations/interactions
- Modular reusable dashboard components
